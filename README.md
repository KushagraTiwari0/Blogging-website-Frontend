# Modern Blogging Platform - Frontend

This is the React frontend for the modern blogging platform. It provides a responsive, dynamic user interface for reading, writing, and managing articles.

## 🌐 Live Demo

**Check out the live website:** [https://blog-0qww.onrender.com/](https://blog-0qww.onrender.com/)


## 🚀 Key Features

- **Modern UI/UX**: Premium, mobile-first design with seamless dark/light mode transitions.
- **Rich Markdown Editor**: Write and format articles using Markdown.
- **Dynamic Avatars**: User-specific avatar generation.
- **Robust State**: React Query for server data caching and Valtio for client state.

## 💻 Tech Stack

- **Framework**: React 18 & Vite
- **Routing**: React Router DOM
- **Data & State**: React Query, Valtio, Axios
- **Markdown Handling**: React Markdown, Turndown (with GFM)
- **Styling**: Vanilla CSS

## 🛠️ Quick Start

1. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

2. Environment setup:
   Create a `.env` file based on `.env.example` and set your backend API URL.

3. Run the development server:
   \`\`\`bash
   npm run dev
   \`\`\`
   The app will be available at http://localhost:5173.
